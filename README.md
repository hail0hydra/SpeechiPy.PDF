# Installation

<br>
<br>

- create a venv and activate it


```shell
python -m venv venv
```

<br>

```shell
source ./venv/bin/activate
```

<br>

- install dependencies



```shell
pip install -r rquirements.txt
```

<br>

- run the program


```shell
python main.py
```
